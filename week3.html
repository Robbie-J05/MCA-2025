Week 3

Key differences: MusicXML vs MEI

1. The first notable difference is that MEI uses "XML ID" for every note that gives it a clear name and corresponding data to identify that particular note. MusicXML contains informational data by telling us exactly where on the page the note is via the use of pixels, but this is very hard to understand as a human reading the code.

2. Another difference is the layout. MEI uses a more layered system where each note is divided with a hierarchical structure that makes information easier for humans to read. MusicXML does not use this system and instead uses more open and closing tags to present information.

3. MEI can detect chords as a whole and has its own "chord" node that MusicXML does not. This provides more recognisable data for humans as the language is more catered to those who are musically informed and can recognise keywords and navigate the score structure more easily.
