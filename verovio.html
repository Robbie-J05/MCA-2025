key differences: Musicxml VS MEI

1. the first notable difference is that MEI uses "XML ID" for every note that gives it a clear name and corresponding data to identify that particular note. music XML contains informational data by telling us exaclty where on the page the note is via the use of pixels but this is very hard to understand as a human reading the code

2. Another difference is the layout, MEI uses a more layered system where each note gets divided up with a hierarchical format that makes informationa little easier to read for humans. Music XML does not use this system and instead ops for a more traditional coding format that uses more open and closing tags to present information.
